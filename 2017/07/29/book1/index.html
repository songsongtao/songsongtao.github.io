开始写博客啦

发表于 2017-07-29 | 分类于 随笔

想到什么写什么

小时候就喜欢拿着笔，在书上、墙上、家具上，留下自己“引以为傲”的字迹，自己也不知道写的是什么，但是就是觉得很有意思，很开心。

长大了整天与电脑为伴，唯一写字的机会也是开会。现在女朋友说起我读书那会写的字还不错，现在就像狗爬一样，这样可不行。就这样，我开始了练字的道路。

但我更喜欢记一点，抄一点。思想和字一起迸发的感觉，让自己沉淀，让自己反思。

最后说一句我爱你俐。

# 鼓励自己 # 有感而发
